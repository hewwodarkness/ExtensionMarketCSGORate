## Встановлення
Це розширення для браузерів на рушії Chromium:
- Розпакувати архів
- Вкладка "Розширення"
- Кнопка "Завантажити розпаковане розширення"
- Вибрати папку з розширенням
- Прикріпити розширення(іконку) на панелі розширень для функцій.
